📊 Power BI Projects
> A collection of end-to-end **Power BI dashboards** and **data analytics projects** showcasing business intelligence, data visualization, and storytelling using real-world datasets.
---
📁 Projects Included
Project	Description
🏪 AdventureWorks Sales Analytics	Sales performance analysis using the AdventureWorks dataset, including revenue trends, regional breakdowns, and KPI tracking
💰 Financial Analysis Report	Interactive financial dashboard covering profit & loss, expense analysis, and financial KPIs
🌍 Global Data Jobs Market Dashboard	Market analysis of data-related job roles, salaries, and demand trends across countries
🏏 Virat Kohli Data-Driven Century Analysis	Sports analytics dashboard analyzing Virat Kohli's century records using cricket statistics
🛒 Walmart Store Sales Analysis	Retail sales analysis covering store performance, seasonal trends, and revenue forecasting
---
🛠️ Tools & Technologies
Tool	Purpose
Power BI Desktop	Dashboard development & visualization

Power Query (M Language)	Data transformation and cleaning
Excel / CSV	Data sources and raw datasets
