# 📊 Power BI Projects

A collection of end-to-end **Power BI dashboards** and **data analytics projects** showcasing business intelligence, data visualization, and storytelling using real-world datasets.

---

## 📁 Projects Included

### 📊 AdventureWorks Sales Analytics  
Sales performance analysis using the AdventureWorks dataset, including:

- Revenue trends  
- Regional breakdowns  
- KPI tracking  

### 💰 Financial Analysis Report  
Interactive financial dashboard covering:

- Profit & Loss  
- Expense analysis  
- Financial KPIs  

### 🌍 Global Data Jobs Market Dashboard  
Market analysis of data-related job roles, including:

- Salaries  
- Demand trends  
- Country-wise insights  

### 🏏 Virat Kohli Data-Driven Century Analysis  
Sports analytics dashboard analyzing:

- Virat Kohli's century records  
- Cricket performance statistics  

### 🛒 Walmart Store Sales Analysis  
Retail sales dashboard covering:

- Store performance  
- Seasonal trends  
- Revenue forecasting
